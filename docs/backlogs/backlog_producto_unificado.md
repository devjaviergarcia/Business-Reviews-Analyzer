# Backlog Unificado del Producto

Este documento concentra las prioridades actuales. Todo el trabajo de producto debe alinearse con estos puntos.

1. Email comercial del análisis (campaña)
Descripción: Diseñar y automatizar un email completo, visualmente sólido y directo, que explique qué hacemos, qué valor entregamos y cómo se interpreta el diagnóstico para captar clientes.

2. Integración de Booking
Descripción: Añadir la fuente Booking al pipeline para descubrir negocio, extraer reseñas, normalizar datos y conectarlo al flujo de análisis y reporte.

3. Integración de Reddit
Descripción: Incorporar Reddit como fuente de menciones/reseñas, con extracción y normalización útiles para diagnóstico reputacional.

4. Integración de Trustpilot
Descripción: Incorporar Trustpilot al pipeline con matching por negocio, scraping estable y datos listos para análisis multi-fuente.

5. Orquestador inteligente de análisis multi-fuente
Descripción: Crear un orquestador que pruebe variantes de nombre por plataforma (por ejemplo: “Negocio”, “Negocio Córdoba”, “Córdoba Negocio”), valide existencia y lance análisis en Google Maps, Tripadvisor, Booking y demás fuentes activas.

6. Landing page del servicio
Descripción: Construir una landing enfocada en conversión, propuesta de valor y captura de leads conectada con el email de campaña.

7. Dashboard de gestión para clientes
Descripción: Desarrollar un panel profesional para clientes con seguimiento de análisis, estado de fuentes, reportes y acciones recomendadas.

8. Profesionalización del LLM y del análisis
Descripción: Elevar la calidad del sistema analítico (prompts, validaciones, estructura de salida, trazabilidad y utilidad de negocio) para que el resultado sea claramente accionable y diferencial.
